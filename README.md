# site-qiutanscore-notes

本仓库用于归档和发布多个独立的 HTML 页面。这些页面以静态文件形式存储，不针对任何特定域名或具体网站，仅作为页面资料的集中存放与版本管理。

## 项目简介

这是一个通用型 HTML 页面归档仓库。仓库内包含多个独立的静态页面，每个页面均可直接通过浏览器打开查看。仓库本身不提供动态服务、后台逻辑或外部数据接口，纯粹用于存储和展示已完成的页面内容。

## 目录说明

```
.
├── pages/          # 存放所有 HTML 页面文件
├── assets/         # 页面引用的静态资源（CSS、JS、图片等）
└── README.md       # 本文件
```

- `pages/`：每个 HTML 文件独立存放，文件名通常反映页面用途或内容主题。
- `assets/`：按需存放页面所需的样式、脚本或媒体文件，与页面文件保持相对路径引用。

## 页面归档说明

- 所有页面均为独立 HTML 文件，不依赖外部服务或数据库。
- 页面之间无相互调用或依赖关系，可单独查看。
- 归档内容可能随时间更新，历史版本可通过 Git 提交记录追溯。

## 维护说明

- 本仓库由维护者不定期更新，新增页面或修正现有内容。
- 欢迎通过 Issue 或 Pull Request 提交建议或修正，但请注意：
  - 不添加任何商业、推广或诱导性内容。
  - 页面内容应保持简洁、清晰，与仓库归档性质一致。
- 维护者保留对页面内容进行筛选和调整的权利。

## 许可

本仓库内容仅供学习、参考与归档使用。具体页面若包含第三方素材，请以页面内标注的许可信息为准。
